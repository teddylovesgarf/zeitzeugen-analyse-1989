Geteilte Erinnerung — Zeitzeugeninterviews zum Mauerfall 1989

Projektarbeit im Kurs *Visualisierung und Analyse von Video und Film*, Universität Leipzig 2026.

Dieses Repository enthält den Code zur Analyse von 20 Zeitzeugeninterviews zum Mauerfall 1989.
Die Interviews wurden mit zwei Methoden analysiert:

- BERTopic; automatische Themenmodellierung auf Basis der Transkripte (satzbasiert)
- ELAN: manuelle Emotionsannotation nach Plutchiks Rad der Emotionen

Ziel ist ein vergleichender Blick auf thematische Schwerpunkte und emotionale Reaktionen
in Abhängigkeit von Erhebungszeitpunkt (vor/nach dem Mauerfall) und gesellschaftlicher
Position der Interviewten.
